Unfinished app with CRUD functionality and RESTful routes. Used MEN (mongo, express, node) stack

I used Semantic2 library to style instead of Bootstrap4

App is about monkeys as who woudn't love animals right? 

Using only basic SOAP principles.

Written in JS.

Instructions: 

1. Install dependencies using command 'npm install'
2. Use command 'node app.js' to run the server
3. Go into the browser and access port localhost:3000
